# 研究规则 / Research Rules

> 明确哪些内容可以自动websearch，哪些必须用户确认，research结果如何归档。

---

## 核心原则

**可以自动research的内容**：
- 市场规模和行业数据
- 政策文件和官方报告
- 竞品公开信息
- 公开案例和获奖项目
- 新闻报道和行业趋势
- 免费可商用的图片素材

**禁止自动编造/脑补的内容**：
- 团队成员真实履历和背景
- 已有合作协议和合作意向
- 已有用户数据和使用反馈
- 已有融资进展和投资方信息
- 项目核心技术细节（需团队确认）
- 商业敏感数据（需团队确认）

## 自动Research触发条件

### 场景1：主档市场规模字段为空
```
触发条件：TAM/SAM/SOM任一字段为"⚠️待补充"
执行动作：
1. WebSearch搜索"[行业]市场规模 2023 2024"
2. 查找权威报告来源（艾瑞/易观/36kr/前瞻研究院）
3. 提取数据，标注来源URL
4. 存入 research/market/行业-market-size.md
```

### 场景2：竞品分析字段不完整
```
触发条件：竞品字段少于2个
执行动作：
1. WebSearch搜索"[行业]头部竞品 竞品分析"
2. 查找产品官网、36kr、IT桔子等来源
3. 提取竞品名称、定位、优劣势
4. 存入 research/competitors/竞品列表.md
```

### 场景3：行业背景需要数据支撑
```
触发条件：主档-行业背景字段为"🔍待验证"
执行动作：
1. WebSearch搜索"[行业]行业报告 2024"
2. 查找政策文件、行业白皮书
3. 提取行业趋势、增长率、政策支持
4. 存入 research/policy/行业政策.md
```

### 场景4：社会价值章节需要支撑
```
触发条件：用户请求写第11章社会价值，且主档无具体量化数据
执行动作：
1. WebSearch搜索"[相关政策]乡村振兴 教育 十四五"
2. 查找政府官网、发改委、教育部文件
3. 提取量化指标（受益人数、资金投入等）
4. 存入 research/policy/社会价值政策.md
```

## Research结果归档规范

### 目录结构
```
research/
├── market/                    # 市场研究报告
│   ├── [行业]-market-size-[日期].md
│   ├── [行业]-user-research-[日期].md
│   └── [行业]-trend-[日期].md
├── policy/                    # 政策文件
│   ├── [行业]-policy-[日期].md
│   └── [相关]-social-value-[日期].md
├── competitors/               # 竞品分析
│   ├── [行业]-competitors-list-[日期].md
│   └── [竞品名]-profile-[日期].md
├── cases/                     # 案例参考
│   ├── [案例名]-case-[日期].md
│   └── 获奖项目参考/
└── image-sources/            #图片来源记录
    └── image-credits-[日期].md
```

### 每份Research文件格式
```markdown
# [文件标题]

## 来源信息
- 来源URL：[URL]
- 来源名称：[如：艾瑞咨询2024年中国XX行业发展报告]
- 访问日期：[YYYY-MM-DD]
- 可信度评级：⭐⭐⭐⭐（4星=权威机构，3星=知名媒体，2星=一般来源）

## 核心数据摘要
[提取的关键数据点，3-5条]

## 数据原文摘录
[引用原文，需标注页码/位置]

## 本项目适用性说明
[说明该数据如何支撑项目主档字段]

## 主档字段更新建议
| 字段 | 原状态 | 更新为 | 更新理由 |
|------|--------|--------|----------|
| TAM | ⚠️待补充 | 🔍待验证 | 引用艾瑞报告数据 |
```

## 数据引用与验证规则

### 数据可信度评级

| 评级 | 来源类型 | 使用建议 |
|------|----------|----------|
| ⭐⭐⭐⭐ | 国家统计局、发改委、权威研究院报告 | 可直接引用 |
| ⭐⭐⭐ | 知名咨询机构（艾瑞、易观、36kr） | 可引用，需标注 |
| ⭐⭐ | 行业媒体、知名公司财报 | 谨慎引用，需交叉验证 |
| ⭐ | 不知名来源、UGC内容 | 不建议引用 |

### 数据引用格式
```
[数据内容]。【来源】[机构名称]，《[报告名称]》，[年份]，[URL]

示例：
2023年中国家庭教育支出同比增长15%。【来源】艾瑞咨询，
《2024年中国家庭教育行业发展报告》，2024年，
https://www.iresearch.com.cn
```

### 估算方法标注
```
当无法获取精确数据时，必须说明估算方法：

TAM估算方法：
[行业总规模] × [目标用户占比] = [TAM]
依据：[行业总规模来自xxx报告，目标用户占比基于xxx调研]

禁止：直接写"估计市场规模为X亿元"而不说明估算依据。
```

## Research流程控制

### 主编决策点
```
用户请求
  ↓
主编判断：是否需要research？
  │
  ├── 不需要 → 直接分配角色写作
  │
  └── 需要 → 判断research类型
              │
              ├── 市场数据 → role-commercial处理
              ├── 竞品信息 → role-commercial处理
              ├── 政策文件 → role-policy处理
              ├── 案例参考 → 主编整合
              └── 图片素材 → role-pitch处理

  ↓
执行research
  ↓
结果存入对应目录
  ↓
更新主档状态（⚠️→🔍）
  ↓
通知用户确认
  ↓
继续章节分配
```

### Research限制
```
单次最多搜索：
□ 市场数据：3个来源
□ 竞品分析：5个竞品
□ 政策文件：2份核心文件
□ 图片素材：每页最多3张备选

超出限制需用户确认。
```

## 禁止事项清单

```
□ 禁止编造市场规模数据
□ 禁止捏造竞品对比数据
□ 禁止虚构政策支持文件
□ 禁止使用未经授权的图片
□ 禁止引用无法验证的数据
□ 禁止在用户确认前将AI推断标记为"已确认"
```
