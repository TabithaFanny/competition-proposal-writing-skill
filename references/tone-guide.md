# 文风统一规则 / Tone Guide

> 所有章节写作必须遵循本规则。角色只影响分析视角，不影响最终成文风格。

---

## 核心理念

**一句话定义**：专业比赛计划书的文风 = "像聪明人在讲真实的事，不是在说服评委"

**三不原则**：
- 不像营销软文（不过度煽情）
- 不像学术论文（不过度晦涩）
- 不像政府报告（不过度空泛）

---

## 成文风格

### 推荐风格特征

| 特征 | 正确示范 | 错误示范 |
|------|----------|----------|
| 专业度 | "2023年家庭教育支出同比增长15%" | "我们的产品非常专业" |
| 简洁度 | "通过NLP技术降低错题整理时间70%" | "我们采用了非常先进的技术手段" |
| 确定度 | "用户调研显示，83%的受访者表示..." | "可能大概也许会有用户喜欢" |
| 证据度 | "相比竞品A，我们响应速度提升3倍" | "我们的产品比竞品好很多" |
| 克制度 | "目标用户为一线城市3-18岁儿童家长" | "我们的用户遍布全国各地" |

### 语气标准
- 用"本项目""项目团队""目标用户"
- 用"我们"指代团队，不用"项目组"
- 用数据表达，不用"非常""特别""极其"
- 用事实表达，不用"颠覆""革命性""爆发式"

---

## 统一表达规则

### 必须统一的术语

| 统一写法 | 禁止写法 |
|----------|----------|
| 本项目/项目团队 | 本产品/我们公司（未注册前） |
| 目标用户 | 用户/客户/消费者（混用） |
| 核心功能名称 | 产品功能/主要功能/亮点功能 |
| 一句话定义 | 愿景/宗旨/理念 |
| 市场规模（TAM/SAM/SOM） | 市场规模/市场容量/市场空间 |
| 融资需求 | 资金需求/融资计划 |
| 当前进展 | 目前状态/现有成果 |

### 段落结构规则

**结论先行**：
```
[结论] + [支撑依据] + [意义/价值]
```

正确示例：
> "儿童教育支出呈现快速增长趋势。根据艾瑞咨询数据，2023年中国家庭教育支出同比增长15%，户均支出超过2万元。这为本项目提供了充足的市场空间。"

错误示例：
> "随着社会发展，教育越来越受到重视。人们对教育的投入越来越大，市场潜力巨大。"

### 数字使用规则
- 百分比必须附绝对值参考
- 时间范围必须明确
- 数据来源必须标注
- 估算必须有方法说明

### 图表引用规则
- 先有结论，再配图表
- 图表为结论服务，不是堆砌数据
- 每张图表必须有解读文字

## 禁止表达清单

### 绝对化用语（P0禁止）
```
颠覆、革命性、爆发式、绝对领先、全球首创、
唯一、第一、最佳、最优、最高、不可替代、
无与伦比、完美、彻底解决
```

### 模糊夸大用语（P1避免）
```
非常、特别、十分、相当、极其、非常之、
巨大、很大、相当可观、未来必将成为
```

### 空洞套话（P1避免）
```
前景广阔、市场潜力巨大、深受用户喜爱、
响应国家号召、符合时代需求、具有重大意义、
为我国xxx事业做出贡献（无具体数据支撑时）
```

### 口语化表达（P2避免）
```
很牛、真的厉害、超棒、碉堡了、YYDS、
小伙伴们、亲们、童鞋们
```

## 角色差异化说明

| 角色 | 分析视角 | 但最终文风必须统一 |
|------|----------|-------------------|
| role-commercial | 市场导向、数据驱动 | ✅ 遵循tone-guide |
| role-product | 技术导向、实现可行 | ✅ 遵循tone-guide |
| role-finance | 增长导向、假设透明 | ✅ 遵循tone-guide |
| role-policy | 价值导向、社会意义 | ✅ 遵循tone-guide |
| role-pitch | 路演导向、一页一意 | ✅ 遵循tone-guide |

**关键原则**：角色只影响"看问题的角度"，不影响"写出来的语气"

---

## 章节风格差异（允许范围）

### 允许的合理差异

| 章节 | 语气倾向 | 原因 |
|------|----------|------|
| 执行摘要 | 高密度、结论极简 | 评委快速扫读 |
| 项目背景 | 偏叙述、有温度 | 建立共情 |
| 产品介绍 | 偏技术、可量化 | 证明可行性 |
| 市场分析 | 数据驱动、冷峻 | 理性说服 |
| 商业模式 | 逻辑清晰 | 商业可行性 |
| 团队介绍 | 专业但有性格 | 展示可信度 |
| 财务分析 | 极克制、数字说话 | 专业信任 |
| 社会价值 | 有情怀但克制 | 避免假大空 |

### 禁止的跨章矛盾

```
第4章说"目标市场为X亿元"
第9章不能写"预计收入Y亿元，Y>>X"

第7章说"团队3人"
第9章不能写"人员成本Z万元（假设10人）"

第2章说"市场增长15%"
第4章不能说"市场增长50%"（除非说明口径不同）
```

## 质量自检清单

每章写完后，作者必须自检：

```
## 文风自检

□ 全文没有"最""第一""绝对"等词
□ 没有"非常""特别""极其"等修饰词
□ 没有"前景广阔""市场潜力巨大"等套话
□ 每段都是"结论+依据"结构
□ 数字都有来源标注或估算说明
□ 没有口语化表达
□ 术语使用统一（查tone-guide术语表）
□ 数据跨章一致（交叉核对）

如有任何一项为"否"，修改后再提交。
```
