### 5. 获取图片的具体步骤

#### Step 1: 确定需要的图片类型
- 背景图：全屏或大区域使用
- 配图：穿插在文字中间
- 图表：数据可视化图片
- 图标：小的功能性图片

#### Step 2: 搜索合适图片
```bash
# 使用搜索引擎搜索免费图片
关键词: [具体场景] free stock photo CC0
```

#### Step 3: 下载和存储
- 格式：优先选择 PNG 或 JPG
- 分辨率：PPT 使用 1920x1080 或更高
- 命名：使用描述性文件名便于管理

## 格式调整指南

### PPT 格式调整

#### 图片统一处理
1. **裁剪**：统一为 16:9 或 4:3 比例
2. **调色**：使用 PPT 自带的色彩校正
3. **抠图**：移除背景，PNG 透明背景
4. **压缩**：PPT 内图片建议压缩至 150dpi

#### 常见问题修复
| 问题 | 解决方法 |
|------|----------|
| 图片模糊 | 替换为更高分辨率版本 |
| 比例失调 | 统一裁剪为同一比例 |
| 风格不统一 | 统一添加滤镜或调整色调 |
| 遮挡文字 | 调整图片位置或添加半透明遮罩 |

#### 跨平台兼容性
- 字体嵌入：确保自定义字体内嵌
- 颜色模式：使用 RGB 模式
- 动画效果：避免过度复杂动画
- 视频嵌入：建议使用云链接而非本地文件

### Word 计划书格式

#### 通用格式要求
| 项目 | 要求 |
|------|------|
| 页边距 | 上下2.54cm，左右3.17cm |
| 正文字体 | 宋体/Times New Roman，12pt |
| 标题字体 | 黑体，16-22pt |
| 行距 | 1.5倍或固定20pt |
| 页码 | 底部居中 |

#### Word 快速样式
1. 标题样式：使用 Word 内置标题样式
2. 目录生成：基于标题样式自动生成
3. 题注：为图表添加自动编号题注
4. 交叉引用：图表引用使用交叉引用功能
